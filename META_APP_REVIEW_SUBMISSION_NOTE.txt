PERMISSION USAGE DETAILS
------------------------

pages_manage_metadata:
- Purpose: Subscribe to page webhooks for receiving Instagram message events in real-time
- Usage: The app uses this permission to subscribe to page webhooks via the Graph API /subscribed_apps endpoint
- User Benefit: Enables real-time Instagram Direct Message reception without polling, providing instant customer support
- End-to-End Experience: 
  1. User grants permission during OAuth flow
  2. App automatically subscribes to webhooks after account connection
  3. Webhook subscription status is displayed in the UI
  4. App receives Instagram messages in real-time via webhooks

pages_read_engagement:
- Purpose: Read page engagement data (posts, followers, likes, comments) to help page managers understand message context and prioritize responses
- Usage: The app uses this permission to read page insights via the Graph API, including follower count and post engagement metrics
- User Benefit: Helps page managers understand their audience engagement, prioritize important messages, and provide context-aware responses
- End-to-End Experience:
  1. User grants permission during OAuth flow
  2. App automatically fetches page insights after account connection
  3. Page insights (followers, recent posts engagement) are displayed in the UI
  4. This data helps page managers understand message context and prioritize responses
